# kg-cli
kg-cli (Kata Gaul command Line Interface) yang digunakan untuk mengetahui dari sebuah kata gaul

## Instalasi
`npm i -g kg-cli`

## Contoh penggunaan
`kg kuy`

Output => 
```bash
user@User ~
$ kg kuy
Kuy
Arti kuy itu kayak ajakan gitu atau sama dengan "Yuk"
A: meet up kuy?
B: Oke, atur tempat aja
A: Kokas deh
```
